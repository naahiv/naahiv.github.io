Files included
- index.html
- styles.css
- script.js

How to use
1. Put all three files in the same directory on your GitHub Pages site.
2. Keep your PDF files in the same directory if you want the relative PDF links to work as written.
3. Edit the `workItems` array in script.js to add, remove, or reorder entries.
4. Edit the short biography/contact text in index.html as needed.
5. If you do not want Google Fonts, remove the font links in index.html and adjust the font-family rules in styles.css.

Notes
- This version uses no build step and no heavy framework.
- The search and filter UI is vanilla JavaScript.
- The layout is responsive and should look much better on both desktop and mobile.
